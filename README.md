# SkBank Recoded

# What is SkBank Recoded?
It's a Minecraft Skript that allows you to add a bank system and economy to your server.

# What you need to use this awesome skript?
skUtilities
Skript (2.2dev37+)
TuSKe
skQuery (for lores)
skript-yaml
